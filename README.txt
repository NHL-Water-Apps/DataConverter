Instructies:

  - Build en exporteer de converter als een .jar file met Eclipse.
  - Roep de .jar file die gemaakt is aan via de command line, met als argument
    het bestand dat je wilt converteren.

    Voorbeeld:
      java -jar VwAppDataConverter.jar inputbestandnaam.gml

  - Voeg de switch '-l' toe om alleen punten in de Leeuwarden regio te pakken

    Voorbeeld:
      java -jar VwAppDataConverter.jar inputbestandnaam.gml -l
